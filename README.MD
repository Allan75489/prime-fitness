# 🏋️ PRIME FITNESS – Sistema de Gestão para Academia

O **Prime Fitness** é um sistema web desenvolvido para auxiliar na gestão de academias.
Ele permite administrar alunos, agendamentos e pagamentos através de um **painel administrativo moderno e organizado**.

Este projeto foi criado com o objetivo de **praticar desenvolvimento web e construção de sistemas administrativos**, simulando um sistema real utilizado em academias.

---

# 📌 Funcionalidades

* 📊 Dashboard administrativo
* 👥 Cadastro e gerenciamento de alunos
* 📅 Sistema de agendamentos
* 💳 Controle de pagamentos
* 📈 Gráfico de novos alunos
* 📱 Interface responsiva
* 🎨 Design moderno estilo dashboard

---

# 🖥️ Telas do Sistema

### 📊 Dashboard

Painel principal com informações gerais do sistema.

* Estatísticas da academia
* Gráfico de novos alunos
* Visão geral do sistema

### 👥 Clientes

Área responsável pelo gerenciamento de alunos.

* Cadastro de novos clientes
* Listagem de alunos
* Controle de informações

### 📅 Agendamentos

Sistema para organizar horários e aulas.

* Controle de horários
* Organização de aulas
* Gerenciamento de agenda

### 💳 Pagamentos

Controle financeiro da academia.

* Registro de pagamentos
* Controle de mensalidades
* Organização financeira

---

# 🛠️ Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript
* Python
* SQLite
* Font Awesome

# 🚀 Como Executar o Projeto

### 1️⃣ Clonar o repositório

```
git clone https://github.com/seuusuario/prime-fitness.git
```

### 2️⃣ Entrar na pasta do projeto

```
cd prime-fitness
```

### 3️⃣ Abrir o sistema

Abra o arquivo:

```
dashboard.html
```

no navegador.

Caso esteja utilizando backend em Python, execute o servidor local antes de abrir o sistema.
